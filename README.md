MX스템을 가진 choc배열의 KLP lame 키캡의 틸트형상이 너무 높아서 오타가 나거나 불편한 사람은 사용을 권합니다.

손가락 우물을 원본보다 더 강조한 설계라서 위치결정이 더 쉽습니다.

높이는 좀 더 낮습니다.

Choc v2와 MX스위치 등 십자 스템을 가진 스위치와의 결합부위를 출력물에 적합하게 조정했습니다.

스위치와 적당한 결합력을 가집니다.

Jlc3dp 의 MJF PA12S 로 출력해서 검증했습니다.
